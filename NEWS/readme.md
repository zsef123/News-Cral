News Analyze Project
==================

## 뉴스 크롤링
각 인터넷 기사의 주 제목, 본문, 날짜(yyyymmdd) 수집

+ 일반 신문 : 조선, 중앙, 동아, 빅카인즈 프로
+ 보건 신문 : 병원신문, 의협신문, 간호신문, 데일리메디, 의사신문, 메다파나뉴스, 메디포뉴스, 헬스코리아뉴스, 메디칼타임즈

* 빅카인즈 프로의 경우 실제 해당 신문사에서 검색결과와 안맞는 경우 있었음( 태어나서 노년까지 틈새없는 인천형 복지사업 추진 2016.9.29 / 간경화는 소변 안 나오면 말기 인정…호스피스 이용 가능 2016.10.27 등)
* 같은 내용의 기사가 중복되는 경우 발생( 신문사 내부적으로는 다른 기사로 처리한다. )
* 사진 주석, 부 제목, Footer 등 본문 이외의 내용이 본문에 섞여 들어가는 경우 발생
* 조선일보의 경우 "완화의료" 등의 키워드로 검색시 "완화" or "의료" 로 검색 결과가 나타난다.

## 형태소 분석
[Konlpy](https://github.com/konlpy/konlpy) Library 사용
Hannanum Class's Morph Method 사용 ( hannanum.morph() ) - [형태소 논의](/etc/morph_check.txt)
원하는 범위 ( 특정 일자 구간, 신문사 별 분류 등)의 기사 본문과 제목의  형태소 개수 각각 파악.

* 단어 Tag는 필요 없기에 표본에 대한 결과만 놓고 보았을 때 다른 클래스들의 분류 결과가 좋지 않았다.

## 네트워크 관계망 
기사 전체에서 랜덤하게 표본을 뽑았을때,
문장의 개수와 문단의 개수가 비슷하다. [Count File](/etc/count_sentence_paragraph.txt)

따라서 가중치 부여는 문단간 출현시 1점으로 한다. 

2D Matrix Form이며 Spreadsheet 파일로 저장 한다.
